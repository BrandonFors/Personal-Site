Keeper App

Basic Description

This is a website I made with a React frontend and Node backend. It allows for someone to keep notes and then delete them when they don't need them anymore. The React client frontend interacts with the Node server backend when a person either adds a note or presses the delete icon. It keeps track of each note component in an array and passes that array to the server through axios so that the information isn't reset on reload.
